# ☕ Management Cafe - Admin Panel & Website

Aplikasi manajemen cafe dengan Flask, MongoDB, dan Bootstrap untuk mengelola website cafe secara lengkap.

## 🚀 Fitur

- **Admin Panel**: Dashboard, Settings, Menu Management, Gallery, Promo, Social Media
- **Frontend Website**: Home, Menu, About, Contact
- **MongoDB Storage**: Semua gambar dan data disimpan di MongoDB (tidak perlu filesystem)
- **Image Compression**: Upload gambar otomatis dikompres
- **Responsive Design**: Mobile-friendly dengan Bootstrap 5

## 📋 Prerequisites

- Python 3.8+
- MongoDB Atlas (Cloud) atau MongoDB Local (localhost:27017)
- pip (Python package manager)
